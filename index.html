Media Queries

DEFINIÇÃO E IMPORTÂNCIA

As media queries são uma técnica utilizada em desenvolvimento web para criar layouts responsivos, ou seja, layouts que se adaptam de forma dinâmica e harmoniosa a diferentes tamanhos de dispositivos, como computadores desktop, tablets e smartphones. Essa técnica é fundamental para proporcionar uma experiência de usuário consistente e agradável, independentemente do dispositivo utilizado para acessar um site ou aplicativo.Com as media queries, é possível aplicar estilos específicos com base em características do dispositivo, como largura da tela, altura da tela, resolução, orientação (horizontal ou vertical), entre outras.

HISTÓRICO E EVOLUÇÃO

As media queries têm uma evolução interessante na história do desenvolvimento web, especialmente no contexto da criação de layouts responsivos. Vamos dar uma breve olhada em como essa técnica evoluiu ao longo do tempo.As media queries foram introduzidas pela primeira vez no CSS3, que é a terceira versão do Cascading Style Sheets, a linguagem utilizada para estilizar páginas web. Isso ocorreu por volta de 2010, como parte dos esforços para tornar os sites mais flexíveis e adaptáveis a diferentes dispositivos. Com o aumento do uso de dispositivos móveis e tablets para acessar a internet, a necessidade de layouts responsivos tornou-se cada vez mais evidente.

EXPLICAÇÃO DOS CONCEITOS FUNDAMENTAIS

Query de Mídia: Uma media query é uma expressão condicional que verifica determinadas características do dispositivo no qual a página está sendo exibida, como largura da tela, altura da tela, orientação, resolução, tipo de dispositivo, entre outros. Com base nessas características, é possível aplicar estilos CSS específicos. Sintaxe : A sintaxe básica de uma media query é composta por uma expressão condicional dentro de um bloco @media. Por exemplo: @media screen and (max-width: 768px) { ... }. Neste caso, o estilo dentro das chaves será aplicado apenas quando a largura da tela for igual ou menor que 768 pixels. Tipos de Mídia: As media queries podem ser direcionadas para diferentes tipos de mídia, como screen (para dispositivos com tela), print (para impressão), speech (para sintetizadores de voz), entre outros. Isso permite personalizar a aparência do conteúdo com base no contexto de exibição.
